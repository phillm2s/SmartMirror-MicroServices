Api call:

Call by coordinates:

weather.html?key={api key}&lon={longitude}&lat={longitude}

Call by city name:

weather.html?key={api key}&city={city name}

Explanation

Variable name Explanation Example
key an verification API key 29hk234734503h45
Chose one from the following options
Option 1: coordinates
lon The longitude of the interested weather area for Bonn: 7.09
lat The latitude of the interested weather area for Bonn: 50.73
Option 2: city name
city Name of the city. To make sure its the right one add the country ID (optional) Bonn or Bonn,DE
